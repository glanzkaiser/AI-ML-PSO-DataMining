FCM.java

Untuk menjalankan file ini dibutuhkan untuk instalasi 
java development kit (jdk). Jdk bisa di download dari
http://java.sun.com

Langkah untuk mengunakan aplikasi :
1. Install JDK
2. Letakkan FCM.java dan CobaFCM.java dalam satu direktori.
3. Compile kedua file tersebut dengan perintah :
    javac FCM.java
    javac CObaFCm.java

4. Jalankan aplikasi dengan perintah :
    java FCM
5. Setiap melakukan simulasi perubahan data, 
   compile ulang file FCM.java dan CobaFCM.java
